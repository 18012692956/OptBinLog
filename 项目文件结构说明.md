# 项目文件结构说明（当前整理版）

## 1. 根目录（论文与总览）

- `毕业论文初稿.md`：论文主文稿（已包含 NanoLog/Zephyr 分析与复现实验结果）
- `测试与实验改进建议.md`：实验改进计划与优先级建议
- `hikbinlog技术文档.md`：Optbinlog 设计与迭代背景文档
- `邓泽宇-2022302111098-武汉大学本科毕业论文（设计）开题报告.doc`：开题报告原件
- `项目文件结构说明.md`：本说明文档

## 2. docs 目录（资料与素材）

- `docs/research/国内外日志系统代码级调研与复现对比.md`：
  国内外日志系统代码级调研、算法复现方案、结果对照
- `docs/assets/flowgraph.jpg`、`docs/assets/iterationgraph.jpg`：论文图示素材

## 3. demo 目录（代码与实验）

### 3.1 核心实现

- `demo/src/`、`demo/include/`：optbinlog 核心 C 实现
- `demo/optbinlog_bench.c`、`demo/optbinlog_multi_bench.c`、`demo/optbinlog_init_race.c`：基准与并发测试程序

### 3.2 实验脚本

- `demo/run_thesis_suite.py`：主论文套件（单机+多设备+竞争+init）
- `demo/run_l1_suite.py`：Linux 多节点真实并发实验
- `demo/run_l1_compare.py`：L1 5/10 节点统计汇总（IQR + mean/median/p95/std/95%CI）
- `demo/run_research_compare.py`：NanoLog/Zephyr 算法复现与语义拉齐对比
- `demo/run_bench.py`、`demo/run_multi_bench.py`、`demo/run_init_race.py`：基础测试脚本

### 3.3 输入数据

- `demo/eventlogst/`：主事件格式与样例日志
- `demo/eventlogst_semantic_min/`：语义拉齐最小事件集（用于 `binary_semantic_like`）

### 3.4 研究复现源码

- `demo/research/research_prototypes.c`：
  `nanolog_like` / `zephyr_deferred_like` 及语义拉齐版本原型实现

## 4. results 目录（结果分组）

- `demo/results/thesis_extnano_20260305/`：本轮扩展主结果（单机高负载、多设备、binary 竞争）
- `demo/results/l1_linux_5_uniform_20260305_extv3/`：5 节点真实并发结果（含 binary/ftrace/nanolog_like/zephyr_deferred_like）
- `demo/results/l1_linux_10_uniform_20260305_extv3/`：10 节点真实并发结果（同上）
- `demo/results/l1_compare_5_vs_10_20260305_extv3.svg`、`..._stats.json`：5/10 节点汇总图表
- `demo/results/research_compare/`：NanoLog/Zephyr 复现与语义拉齐对比结果
- `demo/results/latest`、`demo/results/l1_latest`：主结果软链接
