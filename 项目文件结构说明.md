# 项目文件结构说明

## 1. 根目录

- `毕业论文初稿.md`：论文主文稿（已更新至最新实验结果）
- `测试与实验改进建议.md`：开题报告对齐的实验改进建议
- `hikbinlog技术文档.md`：实现与迭代背景文档
- `flowgraph.jpg`、`iterationgraph.jpg`：流程图素材
- `邓泽宇-2022302111098-武汉大学本科毕业论文（设计）开题报告.doc`：开题报告原件

## 2. demo 目录

- `src/`、`include/`：核心 C 代码
- `eventlogst/`：事件定义与 syslog 原始样本
- `run_*.py`：实验脚本（含双平台混合脚本和一键套件）
  - `run_hybrid_bench.py`：单机双平台
  - `run_hybrid_multi_bench.py`：多设备双平台
  - `run_init_race.py`：竞争初始化（本地）
  - `run_thesis_suite.py`：一键套件入口
- `results/`：统一实验输出目录（按时间戳归档）
